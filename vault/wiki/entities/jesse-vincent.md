# Jesse Vincent

Creator of Superpowers (`obra/superpowers`). Founder of Prime Radiant.

## Background
- Created Request Tracker (RT) in the 1990s — widely-used open-source ticketing system
- Managed Perl 6 development (2005-2008)
- Co-founded Keyboardio (hardware company)
- Built K-9 Mail for Android, later acquired by Mozilla and rebranded as Thunderbird for Android
- The thread: builds infrastructure others rely on, obsesses over workflow

## Contribution to Agent Skills
Created Superpowers in October 2025 as a response to AI coding agents being capable but undisciplined. The framework grew from practical experience using Claude Code for serious development. Identified that agents respond to structure and explicit step-by-step workflows — not verbal best-practice advice.

Key innovations:
- Hard-gate enforcement in skills (not suggestions)
- Subagent-driven development with fresh context per task
- TDD for skills (pressure-test skills with subagents under realistic scenarios)
- Cross-agent skill portability via plain Markdown

Superpowers reached 179K GitHub stars by May 2026 and was accepted into Anthropic's official Claude Code plugin marketplace in January 2026.

## Philosophy
"Treat your AI agent like a powerful but undisciplined junior engineer. Give it the process guardrails that turn juniors into seniors."
